# AntdUI

### 🦄 介绍 Introduce

基于 Ant Design 设计语言的 WinForm UI 界面库，通过将 Ant Design 的现代设计理念与 WinForm 的开发场景深度融合，让桌面应用具备与 Web 应用同级别的视觉品质与用户体验，填补传统 WinForm 应用界面风格陈旧、交互体验单一的空白。

Ant Design for WinForms, which deeply integrates Ant Design's modern design concepts with WinForm development scenarios. This enables desktop applications to achieve visual quality and user experience on par with web applications, filling the gap left by the outdated interface style and singular interaction experience of traditional WinForm applications.
